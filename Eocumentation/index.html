Source File Structure



Fonts Source File structure

Remember to change the font into style.css

            
            <!--All Google Fonts-->
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Montserrat:300,400,500,600,700|Roboto:300,400,500,700">


CSS Source File structure

These are the css files that are loaded into templates in Head Section.


            <!-- CSS here -->
            <link rel="stylesheet" href="assets/css/plugins/all.min.css">
            <link rel="stylesheet" href="assets/css/plugins/flaticon.css">
            <link rel="stylesheet" href="assets/css/plugins/bootstrap.min.css">
            <link rel="stylesheet" href="assets/css/plugins/swiper-bundle.min.css">
            <link rel="stylesheet" href="assets/css/plugins/magnific-popup.css">
            <link rel="stylesheet" href="assets/css/plugins/aos.css">
            <link rel="stylesheet" href="assets/css/plugins/plugins/nice-select.css">
            <link rel="stylesheet" href="assets/css/style.css">
